Ch: 8 Javascript Recursion

Any function which will call itself is said to be recursive.

Recursive solutions are a common way to impliment the "divide-and-conquer" method of problem solving.
For recursion to work, there must be an identifiable base case. That is a circumstance in which it is no longer nessisary to re-call the recursive function. All recursive calls should be moving towards this base case, or else the function may never terminate.

Divide and Conquer

The Divide-and-Conquer method boils down to solving a problem by solving the sub-problems it is composed of.

Tail Recursion

Tail Recursion is a special case of recursion in which the very last thing the function does is call itself. All other calculations and desicitons are performed prior to the single recursive call on the final line.
In theory this allows for recursive functions to take up only a single fram on the stack when the environment supports Proper Tail Calls... In reality only recent versions of Node and Opera.
